Sainsbury’s TechTest

This program renders a link of Sainsbutys grocery shopping website,
and use simple_html_dom class to parse HTML objects and elements.
This program returns parsed data in Json format.

Requirement:

PHP 5.4 or greater is installed.
Composer installed
Dependencies: phpunit, Symphony Console

Installation

https://github.com/nraxz/Sainsburys.git

Running

$ php console.php crawler

Tests

$ phpunit tests
